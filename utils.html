mvacfg.utils.mva_study(name, signame, sigsmp, bkgname, bkgsmp, features, mvatype, mvaconfig=None, outdir='.', methconfig=None, is_sig='is_sig')[source]¶

Main function to perform a MVA study. The results are stored in three different files: one storing the histograms and the ROC curve, another with the configuration used to run this function, and the last stores the proper class to store the MVA algorithm.

Parameters:
  • name (str) – name of the study.
  • signame (str) – signal sample name.
  • sigsmp (pandas.DataFrame) – signal sample.
  • bkgname (str) – background sample name.
  • bkgsmp (pandas.DataFrame) – background sample.
  • features (list of str or None) – list of variables to use.
  • mvatype (MVA classifier) – type of MVA to execute.
  • mvaconfig (dict) – configuration of the MVA.
  • outdir (str) – output directory.
  • methconfig (dict) – configuration of the MVA method.
  • is_sig (str) – name for the additional column holding the signal condition.
Returns:

MVA manager, training and testing samples.

tuple(MVAmgr, pandas.DataFrame, pandas.DataFrame)
